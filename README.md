## 关系配置单是什么？

你好！

如果你也厌倦了整齐划一的社会剧本，欢迎来到 DIY Relationship（关系配置单）。

这是一个可以适用于所有友情和亲密关系、帮助大家探索需求和边界的自助工具。

你需要做的就是：标记自己的态度。

我们编制了10个模块，覆盖97个场景，561个小项，其中一定有你在乎的事情。

我们对你的配置结果**不做任何评价**，只有一条可爱的 Emoji 结果代码，您可以分享给任何人查看，或者**请AI出具分析报告（功能开发中）**。

## 如何答题（重要！）
每一个场景下有若干小项，您可以对每一个小项表态：
- **不表态（跳过不选）**：无所谓/没感觉/暂时不考虑这件事    
- **硬性边界**：绝对不行/分手警告  
- **不确定 /看条件**：可能行，也可能不行，具体情况双方对话沟通
- **同意/接受**：接受/可以，具体程度可商量
- **核心需求**：必需品/快乐源泉

**为什么没有“不行/不喜欢”？ **
- 比起喜欢和不喜欢，我们更在意的是接受和不接受。
- 如果讨厌或者喜欢**不能接受对方与自己相反**的程度，请选“**硬边界**”（红色×）或者“**核心需求**”（金色五角星）
- 如果喜欢/讨厌程度是在中间的灰色地带，请在**不确定/看条件**、和**同意/接受**之间选一个。

## 功能介绍

#### 首页
这里是欢迎页。您可以在这里
- 进入配置单页面
- 粘贴代码自己的代码，继续做题或者直接查看结果
- 粘贴两份代码，对比查看。建议两人同时在场的时候进行对比，这样可以当面认真聊天。

#### 配置单页面 
您需要根据自己的需求选择模块，然后进入做题页面。
- 在这里为你的假想目标选择头像（默认头像是地球，意思是不针对任何人） 。我们对每个人的需求和边界都可能是不一样的。
- 针对你假想的关系选择组合套餐（也可以不选）
- 按下模块卡片可以预览内部题目清单

#### 做题页面
这是本系统的主体，您在这里对您所选的模块中的各个场景表明态度。
- 不选也是态度，可以跳过
- 每页推荐只选1-2个核心需求，太多了没有实际意义
- 随时可以暂停。进度自动保存至浏览器缓存，您也可以在底部手动保存并复制结果代码，之后再回来接续完成。

#### ★ 做题心法
- **区分【愿望】与【底线】**：请点选你现实中 **“实际能接受”** 的程度，而不是自己的愿望。
- **光谱型表态**：有些场景列出的事件，严重程度是层层递进的，你可以按照你的接受范围为它们标记不同的态度，并非只能选择其一。
- **允许内心冲突**：人生不是非黑即白的。某些选项看似互斥，但如果你都很在意，可以都选。

#### 结果页和对比页
这里是您所有结果的展示页、或者两人结果的对比展示页
- 按态度分区，您可以清楚地看到自己或者他人（可以读取任何人的代码）的核心需求和硬性边界等
- 我们为您提供一条Emoji代码，可以发送给任何人查看，也可以自己保存。
- 数据安全提示：我们是一个静态站，不存储您的任何结果。请妥善保存您的结果代码。
- 请注意：这条Emoji代码无容错机制，请确定您保存的代码无误，如误删了其中内容，或者粘贴不完整，系统将显示错误的结果。

#### 右上角菜单
这里是一个简单的控制中心，您可以在这里
- 切换夜间模式
- 切换字号
- 查看 / 删除浏览器缓存中的问卷信息
- 备份数据
- 清空数据
- 返回首页


## 目前没有做的功能
- **帐号系统**：我们是静态站，无法实现此功能。
- **结果页生产长图**：这项功能比较费您的系统资源，而且不同的浏览器对这一功能支持程度不一，待我学习一下，能确保效果再做。
- **短口令**：短口令并不是压缩算法，而是对服务器上保存的资料提供的编号。我们是静态站，无法保存您的资料并对其编号。
- **个性化分析报告**：这需要提前撰写天量的文案，并根据测试数据的细微差异进行排列组合。开发者表示自己精力有限。
- **AI分析**：为了弥补没有分析报告的遗憾，本站会给您生成一份提示词和结果文本，您拿给自己信得过的AI，粘贴进去让它给您生产一份报告。为什么不自己内嵌AI？因为注册国内的AI服务商，拿到它们的API，需要交出我的实名手机号。然后为了防范滥用，您也需要在本站注册，交出您的手机号（收验证码），这太麻烦了，遂放弃。
